# Seekie Pet 项目改进计划

## 1. 代码结构优化
- [ ] 将main.py拆分为模块化结构：
  - `core/` - 核心功能模块
    - `robot.py` - 机器人主类
    - `animation.py` - 动画控制
    - `input_detection.py` - 输入检测
    - `hotkey_manager.py` - 快捷键管理
  - `ui/` - 用户界面模块
    - `window_manager.py` - 窗口管理
    - `tray_manager.py` - 系统托盘管理
  - `config/` - 配置管理模块
    - `config_manager.py` - 配置管理
    - `theme_manager.py` - 主题管理
  - `utils/` - 工具模块
    - `logger.py` - 日志记录
    - `image_loader.py` - 图片加载
