# 桌面宠物机器人 v3.0.1 重构版

一个轻量级的桌面宠物应用，机器人在菜单栏上移动。

## 🎯 重构目标

对3.0.1版本进行代码重构，实现：
1. **简化代码结构** - 移除冗余功能，只保留核心逻辑
2. **清理项目文件** - 删除不必要的工具和测试文件
3. **优化资源管理** - 只保留必要的图片资源
4. **提高可维护性** - 使用常量配置和清晰的代码组织

## 📁 项目结构（重构后）

```
desktop_pet_car/

```

## 🚀 重构改进

### 代码结构优化
- **使用常量配置**：将硬编码值提取为类常量
- **方法命名规范**：私有方法使用`_`前缀
- **逻辑分离**：将复杂功能拆分为独立方法
- **错误处理**：统一的异常处理机制

### 功能简化
- **移除"关于"窗口**：简化系统托盘菜单
- **移除"随机位置"功能**：保留核心移动逻辑
- **简化控制菜单**：只保留显示/隐藏、速度控制、退出

### 性能优化
- **常量配置**：减少运行时计算
- **统一动画循环**：简化移动逻辑
- **资源预加载**：图片加载优化

## 📋 核心功能

### 1. 透明窗口
- 使用Tkinter创建透明窗口
- 黑色背景作为透明色
- 保持在最前面显示

### 2. 菜单栏移动
- 限制在屏幕右半边移动
- 大约每20秒移动一次
- 随机方向和目标位置

### 3. 系统托盘控制
- 右键托盘图标显示控制菜单
- 显示/隐藏机器人窗口
- 调整移动速度
- 退出应用程序

### 4. 图片支持
- 自动加载`Robot_50x50.png`
- 后备图标机制（图片不存在时）
- 自适应托盘图标大小
