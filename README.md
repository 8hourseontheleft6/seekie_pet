# 桌面宠物小车 v2.0.0

一个轻量级的桌面宠物应用，显示一个二轮小车在菜单栏区域移动。

**版本: 2.0.0**  
**最后更新: 2026-04-11**

## 功能特点 (v2.0.0)

- 🚗 **智能移动模式**: 小车大约每分钟移动一次，每次持续5-15秒
- 🎯 **右半边限制**: 小车只在屏幕右半边移动，不干扰主要工作区域
- 👁️ **视角系统**: 
  - 移动时显示一个轮子（侧面视角）
  - 停止时显示两个轮子（正面视角）
- 🎨 **简洁图形**: 小车有车身、车轮、车窗和方向指示箭头
- 📌 **系统托盘集成**: 右键点击可控制应用
- ⚙️ **可调节速度**: 速度范围0.1-2.0（初始0.3）
- 🎯 **随机位置**: 可移动到右半边的随机位置
- 👁️ **显示/隐藏**: 可切换小车窗口的可见性
- 💡 **轻量级**: 资源占用低，优化更新频率

## 安装要求

- Python 3.6 或更高版本
- Windows 系统（也支持 macOS 和 Linux，但菜单栏行为可能不同）

## 安装步骤

1. 克隆或下载本项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

   或者手动安装：
   ```bash
   pip install pillow pystray
   ```

## 使用方法

1. 运行应用：
   ```bash
   python main.py
   ```

2. 应用启动后：
   - 一个红色的小车会出现在屏幕底部，在菜单栏区域来回移动
   - 系统托盘区域会出现一个小车图标
   - 右键点击托盘图标可以打开控制菜单

3. 控制菜单选项：
   - **显示/隐藏**: 切换小车窗口的可见性
   - **速度加快**: 增加小车的移动速度
   - **速度减慢**: 减小小车的移动速度
   - **随机位置**: 将小车移动到屏幕上的随机位置
   - **关于**: 显示关于信息
   - **退出**: 退出应用

## 版本历史

### v2.0.0 (2026-04-11)
- **重大更新**: 完全重新设计移动逻辑
- **右半边限制**: 小车现在只在屏幕右半边移动
- **智能移动**: 大约每分钟移动一次，而不是持续移动
- **视角系统**: 根据移动方向显示不同数量的轮子
- **速度优化**: 降低移动速度，更合理的速度范围

### v1.0.0 (初始版本)

- 系统托盘控制

## 技术实现

- **GUI框架**: Tkinter（Python标准库）
- **系统托盘**: pystray 库
- **图像处理**: Pillow（PIL）库
- **多线程**: Python threading 模块，用于同时处理动画和系统托盘
- **智能算法**: 基于时间的移动触发和方向计算

## 自定义

你可以修改 `main.py` 文件中的以下参数来自定义小车：

- `car_size`: 小车大小（默认32像素）
- 颜色值：可以修改车身、车轮、车窗的颜色
- `speed`: 初始移动速度
- `taskbar_height`: 调整小车距离屏幕底部的距离

## 注意事项

- 在某些系统上，可能需要管理员权限才能正常显示系统托盘图标
- 如果遇到透明色问题，可以修改 `create_window()` 方法中的 `transparentcolor` 参数
- 应用退出时请使用托盘菜单的"退出"选项，以确保所有线程正确关闭

## 故障排除

1. **导入错误**: 确保已安装所有依赖
2. **托盘图标不显示**: 检查系统托盘设置，可能需要重启资源管理器
3. **小车不移动**: 检查控制台输出是否有错误信息
4. **窗口透明问题**: 尝试将 `transparentcolor` 改为其他颜色
