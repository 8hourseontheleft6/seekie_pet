# 桌面宠物小车

一个轻量级的桌面宠物应用，显示一个二轮小车在菜单栏区域来回移动。

## 功能特点

- 🚗 可爱的二轮小车在屏幕底部（菜单栏区域）来回移动
- 🎨 简洁的图形界面，小车有车身、车轮和车窗
- 📌 系统托盘集成，右键点击可控制应用
- ⚙️ 可调节移动速度
- 🎯 可随机移动位置
- 👁️ 可显示/隐藏小车窗口
- 💡 轻量级，资源占用低

## 安装要求

- Python 3.6 或更高版本
- Windows 系统（也支持 macOS 和 Linux，但菜单栏行为可能不同）

## 安装步骤

1. 克隆或下载本项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

   或者手动安装：
   ```bash
   pip install pillow pystray
   ```

## 使用方法

1. 运行应用：
   ```bash
   python main.py
   ```

2. 应用启动后：
   - 一个红色的小车会出现在屏幕底部，在菜单栏区域来回移动
   - 系统托盘区域会出现一个小车图标
   - 右键点击托盘图标可以打开控制菜单

3. 控制菜单选项：
   - **显示/隐藏**: 切换小车窗口的可见性
   - **速度加快**: 增加小车的移动速度
   - **速度减慢**: 减小小车的移动速度
   - **随机位置**: 将小车移动到屏幕上的随机位置
   - **关于**: 显示关于信息
   - **退出**: 退出应用

## 技术实现

- **GUI框架**: Tkinter（Python标准库）
- **系统托盘**: pystray 库
- **图像处理**: Pillow（PIL）库
- **多线程**: Python threading 模块，用于同时处理动画和系统托盘

## 自定义

你可以修改 `main.py` 文件中的以下参数来自定义小车：

- `car_size`: 小车大小（默认32像素）
- 颜色值：可以修改车身、车轮、车窗的颜色
- `speed`: 初始移动速度
- `taskbar_height`: 调整小车距离屏幕底部的距离

## 注意事项

- 在某些系统上，可能需要管理员权限才能正常显示系统托盘图标
- 如果遇到透明色问题，可以修改 `create_window()` 方法中的 `transparentcolor` 参数
- 应用退出时请使用托盘菜单的"退出"选项，以确保所有线程正确关闭

## 故障排除

1. **导入错误**: 确保已安装所有依赖
2. **托盘图标不显示**: 检查系统托盘设置，可能需要重启资源管理器
3. **小车不移动**: 检查控制台输出是否有错误信息
4. **窗口透明问题**: 尝试将 `transparentcolor` 改为其他颜色
