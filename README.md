# 桌面宠物机器人 v3.0.0

一个轻量级的桌面宠物应用，显示一个机器人在菜单栏区域移动。

**版本: 3.0.0**  
**最后更新: 2026-04-11**

## 🎉 重大更新 (v3.0.0)

- 🤖 **全新外观**: 从蓝色小方块升级为Robot.png机器人图片
- 🖼️ **图片支持**: 使用自定义50x50像素机器人图片
- 🚀 **性能优化**: 保持20FPS高帧率流畅动画
- 🔧 **技术修复**: 解决图片路径和Tkinter初始化问题
- 🎯 **界面净化**: 移除移动时的箭头指示器，保持图片干净

## 功能特点 (v3.0.0)

- 🤖 **智能机器人**: 使用Robot.png图片作为桌面宠物
- 🚀 **流畅动画**: 20FPS高帧率，无卡顿感
- 🎯 **右半边限制**: 机器人只在屏幕右半边移动，不干扰主要工作区域
- ⏱️ **智能移动模式**: 机器人每20秒移动一次，每次持续5-15秒
- 📌 **系统托盘集成**: 右键点击可控制应用
- ⚙️ **可调节速度**: 速度范围0.5-5.0（初始1.5）
- 🎯 **随机位置**: 可移动到右半边的随机位置
- 👁️ **显示/隐藏**: 可切换机器人窗口的可见性
- 💡 **轻量级**: 资源占用低，优化更新频率
- 🛡️ **错误处理**: 完整的异常处理和后备方案

## 安装要求

- Python 3.6 或更高版本
- Windows 系统（也支持 macOS 和 Linux，但菜单栏行为可能不同）

## 安装步骤

1. 克隆或下载本项目
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```

   或者手动安装：
   ```bash
   pip install pillow pystray
   ```

## 使用方法

1. 运行应用：
   ```bash
   python main.py
   ```

2. 应用启动后：
   - 一个机器人会出现在屏幕右下角菜单栏区域
   - 系统托盘区域会出现一个机器人图标
   - 右键点击托盘图标可以打开控制菜单

3. 控制菜单选项：
   - **显示/隐藏**: 切换机器人窗口的可见性
   - **速度加快**: 增加机器人的移动速度
   - **速度减慢**: 减小机器人的移动速度
   - **随机位置**: 将机器人移动到屏幕上的随机位置
   - **关于**: 显示关于信息
   - **退出**: 退出应用

## 技术实现

- **GUI框架**: Tkinter（Python标准库）
- **系统托盘**: pystray 库
- **图像处理**: Pillow（PIL）库，支持PNG透明背景
- **多线程**: Python threading 模块，用于同时处理动画和系统托盘
- **智能算法**: 基于时间的移动触发和方向计算
- **路径处理**: 使用绝对路径确保文件可靠访问
- **错误处理**: 完整的异常处理和后备图标系统

## 自定义

你可以通过以下方式自定义机器人：

1. **更换图片**: 替换 `main_pic/Robot_50x50.png` 文件（建议50x50像素）
2. **修改参数**: 在 `main.py` 中调整：
   - `car_size`: 机器人大小（默认50像素，匹配图片尺寸）
   - `speed`: 初始移动速度（默认1.5）
  - `move_interval`: 移动间隔时间（默认20秒）
   - `taskbar_height`: 调整机器人距离屏幕底部的距离

3. **自定义托盘图标**: 机器人图片会自动调整大小作为托盘图标

## 注意事项

- 图片文件必须为PNG格式以支持透明背景
- 在某些系统上，可能需要管理员权限才能正常显示系统托盘图标
- 如果遇到透明色问题，可以修改 `create_window()` 方法中的 `transparentcolor` 参数
- 应用退出时请使用托盘菜单的"退出"选项，以确保所有线程正确关闭

## 故障排除

1. **图片加载失败**: 
   - 检查 `main_pic/Robot_50x50.png` 文件是否存在
   - 程序会自动创建后备图标

2. **导入错误**: 确保已安装所有依赖
   ```bash
   pip install -r requirements.txt
   ```

3. **托盘图标不显示**: 检查系统托盘设置，可能需要重启资源管理器

4. **机器人不移动**: 检查控制台输出是否有错误信息

5. **窗口透明问题**: 尝试将 `transparentcolor` 改为其他颜色
